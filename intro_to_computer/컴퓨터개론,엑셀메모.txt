셀 데이터 삭제

셀 데이터에 delete 키를 누르거나 backspace를 누르면 삭제되다. 

셀 데이터 편집
셀 누르고 수식입력줄에서 편집하면 되다.

셀이 삭제되면서 오른쪽 셀이  왼쪽에 오거나 아래 셀이 위로 올라갈 수 있다.

숫자입력 (1)

단 32이 진짜 32이 아니아 .5가 숨으면서 32로 반올림 되어지다.

숫자입력 (2)

백분율,통화 입력하기

데이터 유효성 검사
[데이터]탭-[데이터 도구]그룹-[데이터 유효성 검사] 클릭
유효성 조건: 제한 하는 값 설정하다
설명메시지: 셀을 선택했을때 제한한 걸 설명해주다.
오류메시지:적합하지 않는 데이터 입력했을때 어떻게 오류됬는지 알려주다.

데이터 유효성 검사-[설정]탭의 제한 대상
-정수:입력 값을 정수로 제한, 제한방법:최소값,최대값
-소수점:입력 값을 실수로 제한,제한방법:최소값,최대값
-목록:1)입력 값으로 무엇 무엇 적으면 셀에서 옆에 목록이 뜬다.
ex) 목록 입력란에 초급,중급,고급,응용 넣으면, 셀에 똑같이 이 네개 항목이 나오다.
2) 셀에 초급,중급,고급,응용을 친다음
 목록을 원하는 셀자리에 데이터 유효성- 원본에서 위 화살표 모양 클릭하고
아까 초급~응용 친 거 드래그 하고 엔터치고 확인누르다
-날짜:입력 값을 날짜/시간 으로 제한, 제한 방법이나 시작 값, 끝 값 설정
-텍스트 길이:입력 텍스트의 길이를 특정 값 또는 최소값, 최대값으로 설정
-사용자 지정:입력 값을 수식 결과 값으로 제한

 하도록 지시하는 미리 작성된 수식

                   >=               크거나 같다               =10>=5                      TRUE
                   <=               작거나 같다               =10<=5                     FALSE

<참조>
한 워크시트 엑셀 값에 다른 워크시트 엑셀 값을 참조하기(3차원 참조)
ex) 한 워크시트 엘셀 값에 다른 워크 시트 한 엑셀값을 참조 해,
 두 값을 빼고 싶은 경우
한 워크시트 엑셀 값에 예를 들어 =R16- 라 치다.
그리고 crtl+pg dn 누르다. 계속 넘어가다. 다른 워크시트로.
그러다가 그 워크시트에 참조하고 싶은  한 엑셀 값 클릭하다.
그리고 다시 pg up 눌러 돌아오다. 확인한 다음 enter 누르다.
그러면 수식 이렇게 되다.
ex)=R16-'2014 수익'!R16
여기서 2014 수익 은 다른 워크시트 이름이다.
 
TIP. 참조의 유형 및 하이퍼링크
* 상대 참조
- '기호 없이 열 머리글과 행 머리글만 사용해서 참조
-상대 참조가 포함된 수식을 복사하면 붙여 넣는 셀 위치에 맞추어 참조가 변경
ex) =A1 (A1이 아닌 다른 곳에서 A1값 불러내다

*절대 참조
-'$'기호를 열 머리글, 행 머리글에 모두 사용해서 참조
-절대 참조가 포함된 수식을 복사하면 어디에 붙여 넣든 참조가 바뀌지 않음
ex) =A1치고 F4번 누르다. 만약 안되면(Fn Lock 해두다.) 
그러면 =$A$1 이렇게 되다

*혼합 참조
-열 머리글이나 행 머리글 한쪽에만 '$'기호를 사용해서 참조
-혼합 참조가 포함된 수식을 복사하면 상대 참조된 부분은 붙여 넣는 셀 위치에 따라
참조가 바뀌고 절대 참조 부분은 바뀌지 않음
ex1)=A1 치고 F4두번 누르다. 그러면 =A$1이 되다.
행만 고정되고 열은 바뀌다
ex2)=A1치고 F4세번누르다. 그러면 =$A1 되다.
행이 바뀌고 열은 고정되다
*3차원 참조
- 다른 워크시트의 이름과 "!"기호를 사용해서 참조

*참조 변경하기
-F4 키를 누르면 상대 참조-> 절대 참조-> 혼합 참조 순으로 변경

*하이퍼링크
- [삽입]탭-[링크]그룹-[하이퍼링크]를 클릭
아니면 링크할 엑셀 칸 선택하고 마우스 우측 클릭,-> 링크 들어가면 되다

<수식에 이름 사용하기>
*한 엑셀 칸 선택-[수식]탭-[정의된 이름]그룹-[이름 정의]클릭
*한 엑셀 칸 선택- 이름 정의 누르다
ex) 이름 설정으로 A부서1분기 B부서1분기 각각 두곳 해놓다.
그리고 다른  엑셀 칸으로 가서 = 이렇게 해놓고
A부서1분기+B부서1분기 이렇게 치면 숫자로 답이나오다.
문자로 치지 않을 경우,[수식]탭-[정의된이름]-[수식에서사용]누르다.

<tip. 이름 정의 규칙>
*첫 번째 문자는 텍스트, _(밑줄) 또는 \(백슬래시)만 사용
*첫 번째 문자를 제외하면 텍스트, 숫자, 마침표, _(밑줄)을 사용 할 수 있고 공백은 사용 할 수 없다.
*c,C,r,R은 예약된 용도가 있으므로 이름으로 사용할 수 없다.
*최대 길이는 255글자다.
*대소문자가 구분되지 않는다.
*한 워크시트에서는 같은 이름을 사용 할 수 없다.
*다른 워크시트라면 같은 이름을 사용해도 괜찮다.

<함수의 구조>
=SUM[A1,A10]
1.등호 =
-함수도 수식의 한 종류이므로
함수를 셀에서 사용할 때는 필요
2. 함수 이름 SUM
-기능에 따른 다양한 함수가 존재
3.괄호 [  ]
-함수 계산에 사용할 값(인수)들은 함수 이름 
다음 괄호 안에 입력
4.인수 A1 A10
-함수에서 계산에 사용하는 값
5.쉼표
-함수 하나 이상의 인수를 사용할 경우 쉼표를 사용하여 인수를 구분함.
(안의 인수 유형을 잘 파악하는 것이 필요)

<통계 함수>
*SUM  함수
-형식: =SUM(인수1,인수2,...)
-예시: =SUM(A1:A2)
*AVERAGE 함수
-형식: =AVERAGE(인수1,인수2,...)
-예시: =AVERAGE(A1:A2)
*MAX / MIN 함수
- 형식: =MAX(인수1, 인수2,...)
          =MIN(인수1,인수2,....)
-예시: =MAX(A1:A4)
         =MIN(A1:A4)
*COUNT 함수
-형식: =COUNT(인수1, 인수2,...)
-예시: =COUNT(A1:A4)
->인수의 개수를 구하는 함수
*RANK 함수
-형식: =RANK(순위를 수하려는 수, 순위를 구하려는 범위, 순위 결정 방법)
-예시: =RANK(A1, A1:A4, 1)
*LARGE/SMALL 함수
-형식: =LARGE(값을 구하려는 범위,순위)->큰 값을 구해주는 함수
         =SMALL(값을 구하려는 범위, 순위)->작은 값을 구해주는 함수
-예시: =LARGE(A1:A4, 2)
         =SMALL(A1:A4,2)

<조건부 함수>
*SUMIF  함수
-형식: =SUMIF(조건범위,조건,합계범위)->조건이 맞는 셀들끼리 합계
-예시: =SUMIF(A2:A6,"야구부",C2:C6)
*SUMIFS 함수(조건이 2개 이상)
-형식: =SUMIFS(합계 범위,조건 범위1, 조건1,조건 범위2,조건2,....)
-예시: =SUMIFS(C2:C5,A2:A5,"연필",B2:B5,"서울")
*AVERAGEIF/AVERAGEIFS 함수
-형식: =AVERAGEIF(조건 범위,조건, 평균 범위)->조건이 맞는 셀들끼리 평균내다
        =AVERAGEIFS(평균 범위,조건 범위1,조건1,조건 범위2,조건2,....)
-예시: =AVERAGEIF(A2:A6,"야구부",C2:C6)
*COUNTIF/ COUNTIFS 함수
-형식: =COUNTIF(범위, 조건)->조건에 맞는 부분만 개수를 세다
        =COUNTIFS(범위1,조건1,범위2,조건2,...)
-예시: =COUNTIF(A2:A7,"연필")

<참조 함수>
*VLOOKUP 함수 -> 상하 방향으로,데이터를 검색해 불러올 값을 찾는 함수
-형식: =VLOOKUP(찾을 값, 범위, 불러올 값이 있는 열 순서, 찾기 옵션)
-예시: =VLOOKUP("A",A2:C5,2,FALSE)
*KHLOOKUP 함수 ->좌우 방향으로,데이터를 검색해 불러올 값을 찾는 함수
-형식: =KLOOKUP(찾을 값, 범위, 불러올 값이 있는 열 순서, 찾기 옵션)
-예시: =HLOOKUP("B", B1:E3,3,FALSE)
*CHOOSE 함수
-형식: =CHOOSE(인덱스 번호,값1,값2,값3,...)-> 셀들에 인덱스 번호값에 따라 값1,값2,값3.. 매긴게 쭉 나열되게 하다, 물론 드래그해서 셀들을 늘려서 하다
-예시: =CHOOSE(A4,"연필","볼펜","지우개","샤프","공책")
*INDEX/MATCH 함수
*형식:=INDEX(셀 범위,행 번호,열 번호) ->셀범위에서 적은 행,열 번호를 찾아서 그 값을 반환하다
        =MATCH(찾을 값,찾을 범위,찾을 방법)->적은 값으로 셀을 찾아 인덱스 행과 열을 알려주다, INDEX와 반대되다
         *찾을 방법 [0:일치, 1: 보다 작은 값, -1:보다 큰 값]
*예시:INDEX(B2:D6,MATCH(B9,A2:A6,0),MATCH(C9,B1:D1,0))    

<Round 함수>
*ROUND 함수  ->반올림
형식:=ROUND(숫자, 반올림할 자릿수(소수이하 몇번째 자리..))
예시:=ROUND(23.157,2)-> 23.16
예시2:=ROUND(-23,-1) ->-20 # -는 반대로 일의자리 십의자리... 올라간다.
예시3:=ROUND(23.5,0) ->24
*ROUNDUP 함수
형식:=ROUNDUP(숫자, 올림 할 자릿수)
예시:=ROUNDUP(23.157,1) ->23.2
*ROUNDDOWN 함수
형식:=ROUND(숫자, 내림 할 자릿수)
예시:=ROUNDDOWN(23.157,1)->23.1

<문자열 함수(1)- 텍스트 변환> -대문자, 소문자 관련 함수
*PROPER / UPPER / LOWER
형식: 
=PROPER(변환할 텍스트 또는 셀 영역): 첫 글자만 대문자로 변환
=UPPER(변환할 텍스트 또는 셀 영역): 모두 대문자로 변환
=LOWER(변환할 텍스트 또는 셀 영역): 모두 소문자로 변환
예시: =PROPER("eXceL") 또는 =PROPER(A2)  -->Excel
*SUBSTITUTE
형식:
=SUBSTITUTE(텍스트 또는 셀 영역, 찾을 텍스트, 바꿀 텍스트, 찾기 옵션)
    -찾기 옵션: 몇 번째 일치하는 텍스트를 변경할 지 지정 (없으면 모두)
예시1:  A2값은  nimcomm,  =SUBSTITUTE(A2,"m","s",2)->nimcosm
예시2:  A3값은  nimcomm,  =SUBSTITUTE(A3,"m","s")->niscoss
예시3:  A4값은  nimcomm,  =SUBSTITUTE(A4,"mm","s")->nimcos

-형식 :=LEFT(텍스트 또는 셀 영역, 추출할 개수)
         =RIGHT(텍스트 또는 셀 영역, 추출할 개수)
-예시 : =LEFT(A2, 2) A2값은  nicoms  ->ni
*MID 함수
- 형식: =MID(텍스트, 시작할 위치, 추출할 개수)

<논리 함수>
*IF 함수
-형식 : =IF(논리식, 값1, 값2)  ->논리식이 참이라면 '값1' 채택하고, 거짓이라면 '값2' 채택하다 
-예시 : =IF(B3>C3,"증가","감소")
*AND 함수
-형식 : =AND(논리식1, 논리식2,...)
-예시: =AND(B3>$B$8, D3="증가") 
 ->논리식1(B3>$B$8), 논리식2(D3="증가") 만족하면 TRUE, 만족하지 않으면 FALSE
*OR 함수
- 형식: =OR(논리식1, 논리식2,...) ->논리식 중 하나만 참이면 TRUE가 되다
- 예시: =OR(B3>$B$8,D3="증가")
*NOT 함수
- 형식: =NOT(논리식) ->논리식이 하나이다 논리식이 참이면 FALSE, 논리식이 거짓이면 TRUE로 하다
- 예시: =NOT(B3>$B$8)

<날짜 및 시간 함수>
*TODAY / NOW 함수
- 형식: =TODAY() / =NOW()
- 예시: =TODAY()   <- 오늘 날짜를 나타낸다.  ->2020-04-30
         =NOW()    <- 오늘 날짜와 현재 시간을 나타낸다.-->2020-04-30 13:41
#시간과 날짜 및 다른 함수 식들을 새로고침 하고 싶으면
-> SHIFT+F9 누르다
# 함수 식을 그대로 쓰고 싶으면?
식 앞에 작은 따옴표( ' )를 붙이다.
'=NOW() -> =NOW() 이렇게 변환되지 않고 바로 나오다.

<TIP. 날짜 및 시간 요소 추출 함수>
EX) =NOW() -> 2020-04-30 14:10
1. =YEAR(날짜): 날짜에서 연도를 추출 -> 2020
2. =MONTH(날짜): 날짜에서 월을 추출  -> 4
3. =DAY(날짜): 날짜에서 일을 추출  -> 30
4. =HOUR(시간): 시간에서 시를 추출 -> 14
5. =MINUTE(시간): 시간에서 분을 추출 -> 10
6. =SECOND(시간): 시간에서 초를 추출 -> 41
7. =WEEKDAY(날짜, 요일순서): 해당 날짜의 요일을 구함 -> 5(월요일을 1이라고 했을때 금요일은 5)
(요일 순서는 1이거나 생략하면 일요일을 1로 시작해서 토요일이 7이 된다.)

*DATE / TIME 함수
- 형식: =DATE(연도, 월, 일) / =TIME(시간, 분, 초)
- 예시: =DATE(2020, 4 ,32)  <- '2020년 5월 2일'로 변환
          =TIME(20, 65, 86) <- '오후 9시 6분 26초'로 변환

<재무 함수>
*PMT(payment) 함수
-일정 금액을 정기적으로 납입하고 일정한 이자율이 적용되는 대출 상황금을 계산
-형식 : =PMT(이율, 상환 기간, 현재 가치, [미래 가치], [지불 시점])  #<- 미래가치, 지불 시점 생략 가능하다
-예시 : =PMT(5%/12,3*12,-5000000)
*PPMT / IPMT (Principal/Interest Payment)함수
- 일정 금액을 정기적으로 납입하고 일정한 이자율이 적용되는 투자에 대한 원금 / 이자의 지급액을 반환
- 형식 : =PPMT(이율, 납부 회차, 상환 기간, 현재 가치, [미래 가치], [지불 시점])
- 예시 : =PPMT(5%/12,5, 3*12, -5000000)
*PV(present Value)함수
- 고정 이자율을 기반으로 대출 또는 투자의 현재 가치를 계산
- 형식 : =PV(이자율, 총 납입 횟수, 납입금, [미래 가치], [지금 시점])
- 예시 : =PV(5%/12, 10*12,-3000000)
*FV(Future Value)함수
- 고정 이자율을 기반으로 투자의 미래 가치를 계산
- 형식 : =FV(이자율, 총 납입 횟수, 납입금, [현재 가치], [지급 시점]) 
- 예시 : =FV(8%/12, 2*12, -5000000)

<Tip.조건부 서식의 종류>
-셀 강조 규칙
:셀에 입력된 값이 특정 값과 같거나 크거나 작거나 범위 사이에 있거나,
특장 텍스트를 포함하거나 중복 데이터가 있을 경우 적용
-상위/ 하위 규칙
:상위 몇 % 이거나 몇 개, 또는 하위 몇 % 이거나 몇 개인 경우, 
평균과 비교해서 크거나 작은 경웨 적용
-데이터 막대
: 셀 값의 크기에 따라 셀을 데이터 막대로 채워서 표시
-색조
:셀 값의 크기에 따라 색과 색의 그라데이션(음영)을 다르게 표시
(2가지 색조를 사용할 경우 최솟값과 최댓값에 다른 색이 사용되고
그 중간에 있는 값들은 두 색이 섞인 정도를 다르게 표시)
-아이콘 집합
:셀 값에 조건 범위를 정하고 해당 조건에 따라 다른 아이콘을 셀에 표시

<Tip. 고급 필터 조건 작성 >

      ="=조건값" 처럼 등호 두 개와 큰따옴표를 사용

5) AND(그리고) 조건: 같은 행의 다른 열에 조건을 입력

<1. 피벗 테이블이란>
-피벗(Pivot)
: 세로축의 각 행에 표시되었던 데이터를 가로축의 각 열에 요약해서 표시되도록 축을 이동시키는 것
-피벗 테이블
: 피벗 기능을 사용해서 만들어진 새로운 표
-피벗 테이블의 특징
1) 대량의 데이터를 여러 가지 방법을 사용해서 요약할 수 있다.
2) 숫자 데이터를 집계해서 부분합을 구하고 항목, 또는 하위 항목별로 데이터를 요약할 수 있다.
3) 데이터 그룹 수준을 확장하거나 축소해서 요약 정보만 표시할 수도 있고 요약된 내용의 세부 데이터를 표시할 수도 있다.
4) 행과 열을 전환해서 데이터를 원하는 형식으로 표시할 수 있다.
5) 원하는 부분에만 필터, 정렬, 그룹, 조건부 서식을 적용하여 강조할 수 있다.
-피벗 테이블의 구성 요소
1) 보고서 필터
2) 필드 머리글 (행, 열)
3) 부분합
4) 총합계

<Tip. 피벗 테이블 필드] 창>
1)보고서에 추가할 필드 선택: 확인란을 선택하여 피벗 테이블에 사용할 필드를 추가
2)도구: 피벗 테이블 칠드 목록의 보기 형식 설정
3)필터: 전체 보고서를 필터링하는데 사용할 필드
4) 열: 열로 사용할 필드
5) 행: 행으로 사용할 필드, 숫자 데이터가 아닌 모든 필드는 일단 행 레이블로 추가됨
6) 값: 값을 표시할 필드. 기본적으로 숫자 데이터를 가진 필드는 값으로 추가됨
7) 나중에 레이아웃 업데이트: 레이아웃을 필드가 선택된 즉시 변경하지 않고 [업데이트]를 클릭할 때까지 기다림
8) 업데이트: 필드 설정에 따라 피벗 테이블 보고서를 갱신

<7. 가상 분석(1) - 시나리오 >
1) [데이터] 탭 - [데이터 도구] 그룹 - [가상 분석] - [시나리오 관리자] 클릭
2) [시나리오 관리자] 대화상자 - [추가] 클릭
3) [시나리오 추가] 대화상자에서 다음을 설정 후 [확인] 클릭
4) [시나리오 값] 대화상자에서 다음을 설정 후 [확인] 클릭
5) 다시 [시나리오 관리자] 대화상자가 나타나면, 같은 방법으로 '이율 7%'부터 '이율 9%'까지의 시나리오를 추가
    -  '이율 7% 일 때의 값: '0.07'
    -  '이율 8% 일 때의 값: '0.08' 
    -  '이율 9% 일 때의 값: '0.09'
6) [시나리오] 항목에서 확인할 시나리오를 선택하고 [표시]를 클릭
7) 해당 적금 이율에 따른 이자 및 만기 시 수령액을 확인

<Tip. 목표값 찾기>
 수식을 사용하여 얻을 결과는 알지만, 결과를 얻기 위해 수식에 입력해야 하는 값을 모를 경우 사용
예) 이자율, 기간, 월 별 대출 상환 가능액을 알고 있을 때 얼마까지 대출을 받으면 상환에 무리가 없는지 확인
 *목표값 찾는 방법
1) [데이터] 탭 - [데이터 도구] 그룹 - [가상분석] - [목표값 찾기] 클릭 
2) [찾는 값]에 원하는 결과 값 입력
3) [값을 바꿀 셀]에 값을 바꾸어 목표값을 찾을 셀에 대한 참조를 입력

<9. 매크로>
1)[3-1] 시트에서 [보기] 탭 - [매크로] 그룹 - [매크로]- [매크로 기록] 클릭 
2) [매크로 기록] 대화상자에서 다음을 설정 후 [확인] 클릭
  - [매크로 이름]: 매크로 이름 입력
  - [바로 가기 키]: 매크로 단축키로 사용할 알파벳을 입력(숫자 사용 불가)
  - [설명] : 해당 매크로의 내용을 입력
3) 매크로 기록이 시작되면, 다음의 작업을 수행
  - A1~E1 셀: [홈] 탭 - [맞춤] 그룹 - [병합하고 가운데 맞춤] 클릭
  - A3~E13 셀: [홈] 탭 - [글꼴] 그룹 - [테두리] - [모든 테두리] 클릭,
		      [맞춤] 그룹 - [가운데 맞춤] 클릭
4) [보기] 탭 - [매크로] 그룹 - [매크로] - [기록 중지] 클릭
5) [3-2] 시트에서 지정한 단축키를 사용하여 매크로가 잘 작동되는지 확인

<Tip. [개발 도구] 탭 추가하기>
1) [파일] 탭 - [옵션] 클릭
2) [Excel 옵션] 대화상자 - [리본 사용자 지정] 클릭
3) 오른쪽의 [리본 메뉴 사용자 지정]항목에서 '개발 도구'에 체크

5) 리본 메뉴에 [개발 도구] 탭이 추가됨

6) 매크로 기록이 시작되면, 다음의 작업을 수행
  - A1~E1 셀: [홈] 탭 - [맞춤] 그룹 - [병합하고 가운데 맞춤] 클릭
  - A3~E13 셀: [홈] 탭 - [글꼴] 그룹 - [테두리] - [모든 테두리] 클릭,
		      [맞춤] 그룹 - [가운데 맞춤] 클릭
